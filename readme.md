# Task Manager API

## Description

A RESTful API built with Node.js and Express.js for managing tasks using in-memory storage.

## Features

- Create, read, update, and delete tasks
- Input validation
- Error handling

### API Endpoints

- GET /tasks
- POST /tasks
- GET /tasks/:id
- PUT /tasks/:id
- DELETE /tasks/:id

## Installation

```
npm install
```

## Running tests

```
npm run test
```

## Running the app.js

```
node app.js
```
